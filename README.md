## 🎯 Mục tiêu bài tập lớn
Dự án này xây dựng một hệ thống Trí tuệ Nhân tạo tích hợp nhằm giải quyết bài toán "Tối ưu lộ trình xe giao hàng trong môi trường giao thông đô thị có yếu tố không chắc chắn (kẹt xe)". Hệ thống áp dụng 5 thành phần cốt lõi: Mạng Bayes, Học máy, Biểu diễn tri thức (Logic), Tìm kiếm Heuristic ($A^*$) và Thỏa mãn ràng buộc (CSP).

## 📂 Cấu trúc thư mục
- `notebooks/`: Chứa Google Colab Notebook chính dùng để demo toàn bộ pipeline hệ thống.
- `modules/`: Chứa mã nguồn Python hỗ trợ (thuật toán, xử lý dữ liệu, mô hình).
- `reports/`: Chứa báo cáo PDF chi tiết về mô hình hóa, thực nghiệm và phân tích.
- `features/`: Chứa các vector đặc trưng, trọng số mô hình đã huấn luyện (.npy, .h5).

## 🚀 Hướng dẫn cài đặt và chạy thử nghiệm
1. Clone repository này: `git clone https://github.com/...`
2. Mở file notebook chính tại `notebooks/main_pipeline.ipynb` bằng Google Colab.
3. Chạy các cell cài đặt thư viện tự động (ví dụ: `osmnx`, `scikit-learn`).
4. Toàn bộ dữ liệu bản đồ sẽ được tải tự động qua public API, không yêu cầu mount Google Drive.
